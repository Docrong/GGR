JBuilder Apache Axis Admin Console

Welcome to the JBuilder Apache Axis Admin Console based on version Axis 1.1

First, configure the project for web services using the Axis toolkit and build the project. Run the project with the Web Services Server runtime configuration to automatically deploy the archives to the Axis server. After you've deployed a web service, you can use the Axis Admin Console to check the state of the Axis server, view the WSDL, view the list of deployed web services, validate the local installation's configuration, and visit the Apache Axis Home Page. Optionally, you can use the SOAPMonitor Applet to view SOAP messages between the client and the server.

Choose one of the following links:

  • View the list of deployed web services
  • Validate the local installation's configuration
  • Administer Axis
    [disabled by default for security reasons ... Enabling instructions]
  • SOAPMonitor
    [disabled by default for security reasons ... Enabling instructions]
  • Visit the Apache Axis Home Page

For more information about how to use the Axis toolkit in JBuilder, see the Web Services Developer's Guide in the JBuilder documentation.
